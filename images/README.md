# Images Folder

Upload your tourism images here following this structure:

## Structure

```
images/
├── logo/
│   └── buenavisit-logo.png (Your logo here)
│
├── hero/                    (Hero carousel images)
│   ├── masao-beach.jpg
│   ├── manlangit-park.jpg
│   └── municipal-plaza.jpg
│
└── places/                  (Place-specific images)
    ├── manlangit-nature-park/
    │   ├── main.jpg
    │   ├── gallery-1.jpg
    │   └── gallery-2.jpg
    │
    ├── masao-public-beach/
    │   ├── main.jpg
    │   └── gallery-1.jpg
    │
    ├── municipal-plaza/
    │   ├── main.jpg
    │   └── gallery-1.jpg
    │
    └── andis-snackbreak/
        ├── main.jpg
        └── gallery-1.jpg
```

## Image Guidelines

- **Logo**: 512x512px, PNG with transparent background
- **Hero Images**: 1920x1080px, JPG, High quality
- **Place Main Images**: 1200x800px, JPG
- **Gallery Images**: 800x600px, JPG
- File size: < 500KB each (optimized for web)

## Placeholder

If you don't have images yet, the system will use placeholder images automatically.
